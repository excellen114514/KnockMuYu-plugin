## 插件安装
```
git clone --depth=1 https://github.com/excellen114514/KnockMuYu-plugin.git ./plugins/KnockMuYu-plugin/
```
> [!TIP]
> 插件并非完全自写，核心代码来自于[面包文游](https://gitee.com/Tloml-Starry/Bread-Shop),仅仅是在其基础简单改写
